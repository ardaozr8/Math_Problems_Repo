## Task 1

The sample space is defined as:
Ω = {ω₁, ω₂, ω₃, ω₄, ω₅}

Events are given as:
- A = {ω₁, ω₃, ω₅}
- B = {ω₂, ω₃, ω₄}

### 1. A ∪ B (Union of events)

The union of events includes all outcomes that belong to **A or B or both**.

A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

A ∪ B = {ω₁, ω₂, ω₃, ω₄, ω₅}

So the union contains **all elements from both sets without repetition**.

---

### 2. A ∩ B (Intersection of events)

The intersection includes only outcomes that are **common to both A and B**.

A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

A ∩ B = {ω₃}

So the only common elementary event is **ω₃**.

---

### 3. B \ A (Difference of events)

The difference B \ A contains outcomes that are **in B but not in A**.

B = {ω₂, ω₃, ω₄}  
A = {ω₁, ω₃, ω₅}

Removing the common element (ω₃):

B \ A = {ω₂, ω₄}

---

### 4. A \ B

The difference A \ B contains outcomes that are **in A but not in B**.

A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

Removing the common element (ω₃):

A \ B = {ω₁, ω₅}
