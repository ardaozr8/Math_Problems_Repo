## Task 3

The monthly cost **u** of running an example laboratory depends on the number **x** of employees hired.  
Let us assume that this dependence is given by the function:

u = 15000x + 10000√x

The number of employees is treated as a random variable **X** with the following distribution:

| xᵢ | 2 | 3 | 4 | 5 |
|----|---|---|---|---|
| pᵢ | 0.10 | 0.25 | 0.40 | 0.25 |

Determine the **probability function of costs (random variable U).**

---

## Step 1: Define the Random Variables

Let:

X = number of employees hired

The cost function is defined as:

U = 15000X + 10000√X

Since **X is a random variable**, the cost **U** is also a random variable.

---

## Step 2: Compute Cost Values

Calculate **U** for each possible value of **X**.

### For X = 2

U = 15000(2) + 10000√2  
U = 30000 + 14142.14  
U ≈ 44142.14

### For X = 3

U = 15000(3) + 10000√3  
U = 45000 + 17320.51  
U ≈ 62320.51

### For X = 4

U = 15000(4) + 10000√4  
U = 60000 + 20000  
U = 80000

### For X = 5

U = 15000(5) + 10000√5  
U = 75000 + 22360.68  
U ≈ 97360.68

---

## Step 3: Probability Function of U

Since **U depends on X**, the probabilities remain the same.

| X | P(X) | U = 15000x + 10000√x |
|---|---|---|
| 2 | 0.10 | 44142.14 |
| 3 | 0.25 | 62320.51 |
| 4 | 0.40 | 80000 |
| 5 | 0.25 | 97360.68 |

---

## Probability Distribution of U

| U (Cost) | Probability |
|---|---|
| 44142.14 | 0.10 |
| 62320.51 | 0.25 |
| 80000 | 0.40 |
| 97360.68 | 0.25 |

---

## Conclusion

The **probability function of the cost random variable U** is obtained by applying the cost function

U = 15000X + 10000√X

to each value of **X** and keeping the same probabilities from the distribution of **X**.
