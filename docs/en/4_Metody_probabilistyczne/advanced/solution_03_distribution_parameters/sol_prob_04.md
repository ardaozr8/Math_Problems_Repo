# Task 4 — Solution

The density function is:

f(x) = 1/2, for 0 ≤ x ≤ 2

---

This is a uniform distribution on the interval [0, 2].

---

## Mode

Since the density is constant (1/2) for all values between 0 and 2, every value in this interval has the same probability density.

So there is no single highest point.

**Mode: any value in [0, 2]**

---

## Median

The median is the value that splits the area into two equal parts.

Since the distribution is uniform from 0 to 2, the middle point is:

Median = (0 + 2) / 2 = **1**

---

## Final answers:

- Mode: any value in [0, 2]  
- Median: **1**
