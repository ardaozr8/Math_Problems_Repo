# Task 5 — Solution

To compare which feature is more stable, I calculate the coefficient of variation for both variables:

v = σ / E

---

For height X:

v_X = 5 / 170 ≈ 0.0294

So the coefficient of variation for height is about **2.94%**.

---

For weight Y:

v_Y = 5 / 65 ≈ 0.0769

So the coefficient of variation for weight is about **7.69%**.

---

Since the coefficient of variation is smaller for height, height is more stable.

## Final answer:

- Height: **2.94%**
- Weight: **7.69%**
- **Height is more stable**
