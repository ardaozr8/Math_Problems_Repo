# Task 7 — Solution

The time between calls follows an exponential distribution with parameter λ = 2.

---

## 1. Expected value

For an exponential distribution:

E(X) = 1 / λ

So:

E(X) = 1 / 2 = **0.5 minutes**

---

## 2. Probability P(X < 3)

For exponential distribution:

P(X < t) = 1 - e^(-λt)

So:

P(X < 3) = 1 - e^(-2 · 3)  
P(X < 3) = 1 - e^(-6)

e^(-6) ≈ 0.00248

P(X < 3) ≈ 1 - 0.00248 = **0.99752**

---

## Final answers:

- E(X) = **0.5 minutes**
- P(X < 3) ≈ **0.9975**
