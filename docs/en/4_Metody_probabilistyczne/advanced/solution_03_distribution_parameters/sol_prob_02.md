# Task 2 — Solution

## Given:

U = 15000X + 10000√X

| x_i | 2 | 3 | 4 | 5 |
|-----|---|---|---|---|
| p_i | 0.10 | 0.25 | 0.40 | 0.25 |

---

## Step 1. Compute U(x) for each value

For X = 2:  
U(2) = 15000·2 + 10000√2 ≈ 30000 + 14142.14 = **44142.14**

For X = 3:  
U(3) = 15000·3 + 10000√3 ≈ 45000 + 17320.51 = **62320.51**

For X = 4:  
U(4) = 15000·4 + 10000√4 = 60000 + 20000 = **80000**

For X = 5:  
U(5) = 15000·5 + 10000√5 ≈ 75000 + 22360.68 = **97360.68**

---

## Step 2. Expected value E(U)

E(U) = Σ U(x_i) · p_i

E(U) = (44142.14)(0.10) + (62320.51)(0.25) + (80000)(0.40) + (97360.68)(0.25)

E(U) = 4414.21 + 15580.13 + 32000 + 24340.17

E(U) ≈ **76334.51**
