# Task 6 — Solution

Each day is a Bernoulli trial (either the limit is not exceeded or it is exceeded), so this is a **binomial distribution**.

X ~ Bin(n = 5, p = 0.8)

---

## 1. Probability formula

P(X = k) = C(5, k) (0.8)^k (0.2)^(5 - k)

---

## 2. Expected value and variance

E(X) = n p = 5 · 0.8 = **4**

D²(X) = n p q = 5 · 0.8 · 0.2 = **0.8**

---

## Final answers:

- Distribution: **Binomial**
- E(X) = **4**
- D²(X) = **0.8**
