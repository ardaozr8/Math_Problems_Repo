# Task 9 — Solution

To calculate the linear correlation coefficient, I first compute the expected values of X and Y.

---

## 1. Expected values

### Expected value of X

E(X) = 8·0.15 + 9·0.25 + 10·0.40 + 11·0.20

E(X) = 1.2 + 2.25 + 4.0 + 2.2 = **9.65**

### Expected value of Y

E(Y) = 1.2·0.14 + 1.3·0.36 + 1.4·0.35 + 1.5·0.15

E(Y) = 0.168 + 0.468 + 0.49 + 0.225 = **1.351**

---

## 2. Second moments and variances

### E(X²)

E(X²) = 8²·0.15 + 9²·0.25 + 10²·0.40 + 11²·0.20

E(X²) = 64·0.15 + 81·0.25 + 100·0.40 + 121·0.20

E(X²) = 9.6 + 20.25 + 40 + 24.2 = **94.05**

So:

D²(X) = E(X²) - (E(X))²

D²(X) = 94.05 - (9.65)²  
D²(X) = 94.05 - 93.1225 = **0.9275**

---

### E(Y²)

E(Y²) = 1.2²·0.14 + 1.3²·0.36 + 1.4²·0.35 + 1.5²·0.15

E(Y²) = 1.44·0.14 + 1.69·0.36 + 1.96·0.35 + 2.25·0.15

E(Y²) = 0.2016 + 0.6084 + 0.686 + 0.3375 = **1.8335**

So:

D²(Y) = E(Y²) - (E(Y))²

D²(Y) = 1.8335 - (1.351)²  
D²(Y) = 1.8335 - 1.825201 = **0.008299**

---

## 3. Mixed moment E(XY)

E(XY) = Σ x·y·p(x,y)

E(XY) = 8·1.2·0.10 + 9·1.2·0.04 + 8·1.3·0.05 + 9·1.3·0.11 + 10·1.3·0.20 + 9·1.4·0.10 + 10·1.4·0.15 + 11·1.4·0.10 + 10·1.5·0.05 + 11·1.5·0.10

E(XY) = 0.96 + 0.432 + 0.52 + 1.287 + 2.6 + 1.26 + 2.1 + 1.54 + 0.75 + 1.65 = **13.099**

---

## 4. Covariance

cov(X,Y) = E(XY) - E(X)E(Y)

cov(X,Y) = 13.099 - (9.65)(1.351)

cov(X,Y) = 13.099 - 13.03715 = **0.06185**

---

## 5. Correlation coefficient

ρ = cov(X,Y) / √(D²(X)·D²(Y))

ρ = 0.06185 / √(0.9275 · 0.008299)

ρ ≈ **0.705**

---

## Final answer:

**ρ ≈ 0.705**
